Chrome TTS Setup Guide

Chrome browser တွင် အသံဖတ်မရပါက အောက်ပါအတိုင်း ပြင်ဆင်ပါ။

၁။ Google Speech Services သွင်းခြင်း

Play Store တွင် "Speech Recognition and Synthesis from Google" ကိုရှာပြီး Install သို့မဟုတ် Update လုပ်ပါ။

၂။ Text-to-speech Menu ရှာဖွေခြင်း

အောက်ပါနည်းလမ်း (၂) မျိုးထဲမှ အဆင်ပြေရာကို သုံးနိုင်ပါသည်။

နည်းလမ်း (က): Search အသုံးပြုခြင်း
Settings အပေါ်ဆုံး Search bar တွင် text to speech ဟုရိုက်ရှာပါ။
နည်းလမ်း (ခ): Manual ရှာဖွေခြင်း
Go to: Settings > Additional Settings > Languages & input > Text-to-speech output.

၃။ Preferred Engine ရွေးချယ်ခြင်း

Preferred Engine ကိုနှိပ်ပြီး "Google Speech Services" ကိုရွေးပါ။

⚠️ သတိပြုရန်: ရွေးချယ်ပြီးပါက Recent Apps ထဲမှ Settings ကိုအပြီးပိတ် (Remove) လိုက်ပါ။ ပြီးမှ ပြန်ဖွင့်ဝင်ကြည့်လျှင် Google Preferred ဖြစ်နေသည်ကို မြင်ရပါမည်။

၄။ Voice Data နှင့် Language သတ်မှတ်ခြင်း

Preferred engine ဘေးရှိ Arrow (>) ကိုနှိပ်ပြီး Install voice data ထဲသို့ဝင်ပါ။

UK နှင့် US English Voice pack (၂) ခုလုံးကို Download ဆွဲထားပါ။

ဒေါင်းလုဒ်ပြီးပါက Language setting တွင် English (United Kingdom) ကိုသာ ရွေးချယ်ပေးပါ။

အားလုံးပြီးပါက စစ်ဆေးရန်:

  • Google Speech Services ကို Install လုပ်ပြီးပြီ။
  • Preferred Engine ကို Google ဟု ရွေးပြီးပြီ။
  • Language ကို English (UK) ဟု ပြောင်းပြီးပြီ။
